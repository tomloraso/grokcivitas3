# 10A - SEO Location Pages

## Goal

Create indexable location or postcode-area landing pages that expose stable, source-backed school research content without depending on premium-only flows.

## Scope

- define page types and URL strategy
- decide which content can be static, cached, or generated ahead of time
- expose canonical metadata and search-engine-friendly structure

## Guardrails

- SEO pages must remain factual and non-editorial.
- Premium-only data should not leak into public indexable pages.
- Generated content must reuse existing serving contracts rather than duplicating business logic in the frontend.

## Acceptance Criteria

- Page types and generation strategy are defined.
- Required API or build-time data shape is documented.
- Search visibility work stays aligned with current product boundaries.
